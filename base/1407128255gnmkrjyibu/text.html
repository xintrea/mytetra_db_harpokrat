Для работы с SPI в Raspberry Pi можно использовать библиотеку wiringpi.


Устанавливаем библиотеку wiringpi по инструкции с официального сайта:



https://projects.drogon.net/raspberry-pi/wiringpi/download-and-install



Чтобы программа собиралась с этой библиотекой, необходимо ее указать в команде компиляции:



gcc -o “%e” -l/usr/local/include “%f” -L/usr/local/lib -lwiringPi



Где опция -lwiringPi - это и есть подключение библиотеки.


Вот более короткий пример для компиляции программы из одного си-файла test.c:



cc -o test test.c -lwiringPi



Если возникнут проблемы при использовании этой библиотеки, можно на вышеуказанной странице почитать комментарии (их там 320 штук).
