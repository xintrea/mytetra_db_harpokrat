Добавить в меню Target - Debug значение:


'Simulator section' CPU DLL "-REMAP"


Далее надо правильно инициализировать память. Для этого добавляем в *.ini файл следующие строки:



MAP 0x00000000,0x0000FFFF READ EXEC   

MAP 0x08000000,0x0801FFFF READ EXEC

MAP 0x20000000,0x200FFFFF READ WRITE EXEC   

MAP 0x40000000,0x400FFFFF READ WRITE EXEC   



Если *.ini файла нет, то создаем его сами и даем на него ссылку в диалоговом окне "Target Options" на вкладке Debug.
